# Tester's Game Switch

A mod for Minecraft.  
Adds shortcut keys to frequently used commands in debugging.

![sample](https://raw.githubusercontent.com/lichtmec/TesterGameSwitch/master/doc_contents/sample.gif)

## Controls

|Default Key|Behavior|
|---|---|
|**Insert**|Switch gamemode : Survival <-> Creative|
|**Delete**|Switch gamerule : *doDaylightCycle* and *doWeatherCycle*<br><br>\* MC1.7.10 does not support *doWeatherCycle*. So, *doWeatherCycle* will only work if the gamerule are added by other add-ons.|
